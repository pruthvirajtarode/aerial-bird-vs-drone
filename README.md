# Aerial Bird vs Drone - Internship-ready Project (Option A)

This is the final submission package (Option A) prepared for internship submission.
Datasets are NOT included in this ZIP. Place the datasets in the project root as:

- classification_dataset/  (train/valid/test with bird/drone folders)
- object_detection_Dataset/ (images + YOLO .txt labels)

Quick start (Windows):

1. cd to project folder
2. python -m venv venv
3. .\venv\Scripts\activate
4. pip install -r requirements.txt
5. python src/create_dummy_model.py
6. streamlit run streamlit_app/app.py
